# Academy: Angular in Action

Welcome to the **Angular in Action** workshop repository! This project is designed to provide a hands-on learning experience for the Angular framework, moving from fundamental concepts to advanced patterns.

## Course Structure

The repository is structured to support two levels of learning:

### 1. Basic Course

- **Starting Point:** [`main`](https://github.com/chumachun/academy-ng-in-action/tree/main) branch.
- **Goal:** Learn the core building blocks of Angular.
- **Solution:** Found on the [`solution/basic`](https://github.com/chumachun/academy-ng-in-action/tree/solution/basic) branch.

### 2. Advanced Course

- **Starting Point:** [`solution/basic`](https://github.com/chumachun/academy-ng-in-action/tree/solution/basic) branch.
- **Goal:** Deep dive into advanced state management, performance, and architectural patterns.
- **Solution:** Found on the [`solution/advanced`](https://github.com/chumachun/academy-ng-in-action/tree/solution/advanced) branch.
