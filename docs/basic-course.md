# Basic Course

The Basic Course covers the fundamental building blocks of Angular.

## Learning Objectives

- Component architecture
- Templates and Data Binding
- Directives and Pipes
- Services and Dependency Injection
- Basic Routing
- Handling User Input

## Workshop Materials

- **Starting Branch:** `main`
- **Solution Branch:** `solution/basic`
- **Slides:** (at request)

## Exercises

1. **Introduction:** Understanding the project structure.
2. **Components:** Creating and nesting components.
3. **Data Flow:** Using `input()` and `output()`.
4. **Services:** Sharing data and logic.
