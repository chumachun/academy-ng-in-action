# Advanced Course

The Advanced Course takes your Angular skills to the next level by focusing on professional patterns and performance.

## Learning Objectives

- Advanced RxJS patterns
- Reactive State Management (NgRx/Signals)
- Custom Directives and Pipes
- Advanced Routing (Guards, Resolvers, Lazy Loading)
- Interceptors
- Performance Optimization

- **Starting Branch:** `solution/basic` (The end result of the basic course)
- **Solution Branch:** `solution/advanced`
- **Slides:** (at request)

## Exercises

1. **State Management:** Migrating to a reactive state pattern.
2. **Interceptors:** Handling global HTTP concerns.
3. **Advanced Routing:** Protecting routes and pre-fetching data.
4. **Performance:** Optimizing change detection and bundle size.
